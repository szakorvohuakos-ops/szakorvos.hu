# Szakorvos.hu

Magyar orvoskereső platform AI asszisztenssel.

## Fájlok

- `index.html` – Főoldal (landing)
- `talalatok.html` – Találati oldal
- `tudastar.html` – Egészségügyi tudástár A-Z
- `admin.html` – Szuperadmin felület
- `admin-orvos.html` – Orvos admin felület
- `admin-klinika.html` – Klinika admin felület

## Tech stack

- Frontend: Vanilla HTML/CSS/JS
- Backend: Supabase (PostgreSQL + Edge Functions)
- AI: OpenAI GPT-4o-mini (Supabase Edge Function-ön keresztül)
- Hosting: Vercel

## Deploy

1. GitHub repoba töltsd fel a fájlokat
2. Vercel.com → Import Git Repository
3. Framework: Other
4. Deploy!
